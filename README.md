# AI Business Readiness Assessment

An interactive scoring tool that evaluates AI implementation readiness across three dimensions: data infrastructure, team capabilities, and organizational culture.

**Try it live → [mattritterspach.com/ai-assessment](https://www.mattritterspach.com/ai-assessment)**

## What It Does

- 12 questions across 3 sections
- Weighted scoring with tier classification
- Personalized recommendations by dimension
- Automated report delivery via EmailJS to both respondent and consultant

## Readiness Tiers

| Score | Tier | What it means |
|-------|------|---------------|
| 0–30% | Early Stage | Significant groundwork needed |
| 31–55% | Building | Strong potential, key gaps |
| 56–75% | AI Ready | Ready for meaningful pilots |
| 76–100% | AI Leader | Positioned to lead with AI |

## Sections

1. **Data Infrastructure** — Quality, centralization, governance, historical availability
2. **Team Capabilities** — AI expertise, leadership maturity, capacity, pilot experience
3. **Organizational Culture** — Automation openness, executive buy-in, adoption speed, budget

## Tech Stack

- Pure HTML/CSS/JavaScript — zero framework dependencies
- EmailJS for automated report delivery
- Deployed on Squarespace via Code Block embed
